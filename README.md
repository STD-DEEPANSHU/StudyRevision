# 🤖 Auto Revision Bot

Telegram bot that:
- Reads PDF / Website / Text
- Generates Q&A using AI
- Sends auto revision messages every few hours
- Repeats questions for better memory

## Setup

1. Add bot token in config.py
2. Install requirements:
   pip install -r requirements.txt
3. Run:
   python bot.py
